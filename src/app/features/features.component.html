Características comunes para todos los productos web

Todos los productos web dan como resultado:

- Páginas web optimizadas para posicionamiento en buscadores, de serie, gracias a direcciones web amigables, control total sobre títulos de página y metacontenido (meta tags), sitemap integrado y actualizado dinámicamente, o su código optimizado y compactado para la máxima velocidad de navegación, entre otras características.
- Páginas web estándar, utilizando la tecnología más compatible para visualizarse correctamente en los navegadores más utilizados y dispositivos móviles.
- Páginas web dinámicas, con núcleo modular que permite extender su funcionalidad según la necesidad, sin límite de número de páginas o archivos (sólo según capacidad del alojamiento web), e incluyendo en todos los casos gestor de contenidos.
- Páginas web completas, porque desarrollar un sitio web no sólo es dibujarlo. Asistencia técnica para el posicionamiento en buscadores, elaboración de contenidos, traducción de textos, etre otros.
